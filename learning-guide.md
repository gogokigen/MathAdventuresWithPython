# Learning Guide

我們會把重心放在「學習數學」上。也就是說，在這份筆記裡，Python 是個輔助工具，而非學習目的。因此，我們會善用 Python 已有的模組和工具，例如：Processing，去畫出我們要的數學函數圖或模擬圖。\
\
若你的興趣著重在：如何寫出這種繪圖程式，那我會建議你去尋找 Python 程式設計相關的教材。\
